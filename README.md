在洒满阳光的阳台上看书的

而我仍旧仰望星空的日子

是和毛
在大千庄园
荡着秋千
无所不谈
忘记了时间

我想念的
是和韵
在七宝古镇
闲庭信步
任小桥流水

是和茜
在丽娃河畔
观水面微波涟漪

所有温暖而容易被人遗忘的瞬间
